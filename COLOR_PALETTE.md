# Color Palette

## Primary Colors

### Purple (Primary Brand Color)
- **Primary Purple**: `#5416b6` - Main brand color, used for primary buttons, links, and key UI elements
- **Primary Purple Dark**: `#3f1089` - Darker shade for hover states and emphasis
- **Primary Purple Light**: `#ede5f7` - Light tint for backgrounds and subtle highlights

## Secondary Colors

### Coral (Secondary/Accent)
- **Secondary Coral**: `#ff6b6b` - Warm, inviting accent color
- **Secondary Coral Dark**: `#e85555` - Darker shade for emphasis
- **Secondary Coral Light**: `#ffe5e5` - Light tint for backgrounds

## Accent Colors

### Amber (Warm Accent)

## Usage Guidelines

### Primary Purple
Use for:
- Primary call-to-action buttons
- Main navigation links
- Important headings and highlights
- Brand elements

### Coral & Warm Accents
Use for:
- Secondary buttons and actions
- Decorative elements
- Feature highlights
- Gradients combined with primary purple

### Neutral Colors
Use for:
- Body text (Neutral 900, 800)
- Secondary text (Neutral 600, 500)
- Borders and dividers (Neutral 200, 300)
- Backgrounds (Neutral 50, 100, White)

## Color Combinations

### Recommended Pairings
1. **Primary Purple + Coral**: Creates energy and warmth
2. **Primary Purple + Amber**: Professional yet inviting
3. **Coral + Rose**: Soft, comforting combination
4. **Purple + Neutral 50**: Clean, modern look

### Gradients
- Purple to Coral: `linear-gradient(135deg, #5416b6, #ff6b6b)`
- Amber to Rose: `linear-gradient(135deg, #f59e0b, #fb7185)`
- Light backgrounds: Use light tints with neutral white
